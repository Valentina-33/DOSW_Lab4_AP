**¿Qué es un arquetipo (Archetype) en Maven?**

Es una plantilla de proyecto que tiene una estructura y configuración base. Este se encuentra disponible en un documento llamado pom.xml.

Tiene componentes esenciales como el main y el test.

Es decir, es un modelo que nos propone una estructura estándar.


**¿Para qué sirve el arquetipo maven-archetype-quickstart?**

Es el arquetipo más simple y básico que contiene una clase principal llamada App.java y una de prueba AppTest.java, y sirve para:

 - Aprender Maven.

 - Hacer proyectos pequeños.

 - Crear ejercicios de clase.

 - Probar configuraciones.

 - Evitar el uso de frameworks pesados.

**¿Cuál es el comando con el cual se puede crear un proyecto basado en**

**un arquetipo maven?**

El comando es:

mvn archetype:generate

Ese comando ejecuta el plugin de arquetipos de Maven y permite generar un nuevo proyecto a partir de una plantilla previamente definida.



**¿Qué es un pull request en GitHub?**


Es una petición que se realiza para hacer merge con una rama y que esta misma se elimine cuando sea aceptada por algún integrante.


**¿Cómo se crea un pull request?**

Un pull request se puede crear directamente del repositorio de GitHub, en donde le damos a la parte superior Pull Request, y luego seleccionamos las ramas en las que queremos hacer pull request.

Después de esto, le damos commit changes y esperamos a que algún otro integrante le de aceptar al Pull request con comentarios anexados. 


**¿Cómo se aprueba un pull request en GitHub?**

1. Entrar al repositorio

Ir a la pestaña Pull requests.

Seleccionar el PR que quieres revisar.

2. Revisar los cambios

Ir a la pestaña Files changed.

Ver qué archivos fueron modificados.

Puedes dejar comentarios línea por línea si algo necesita ajuste.

3. Agregar una revisión

Hacer clic en Review changes.


Comment → Solo dejar observaciones.

Approve → Aceptar los cambios.

Request changes → Pedir correcciones antes de aceptar.

4. Hacer merge

Si todo está correcto y el PR está aprobado:

Hacer clic en Merge pull request.

Confirmar el merge.
